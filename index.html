üîë 5 Rules to Internalize

  • Be explicit: "Write a 3-paragraph executive summary for a non-technical VP" beats "summarize this"
  • Give context & motivation: "This goes to the board, so accuracy and tone matter more than speed"
  • Show examples: Paste in past work you liked and say "match this style and depth"
  • Specify format: "Use headers. Bullet the risks. Keep it under 500 words."
  • Iterate, don't restart: If output is 70% right, tell Claude what to fix. Don't re-prompt from scratch.

üõ†Ô∏è Do It (30 min)

Day 8 ‚Äî Wednesday
ü§ù Cowork ‚Äî Your AI Coworker

‚úÖ Platform note: Cowork works on both Windows and Mac with full feature parity. Open the Claude Desktop app ‚Üí click "Cowork" at the top. Same experience on both platforms.

Why: Cowork is the biggest unlock in this bootcamp for non-developers. You point Claude at a folder on your computer and it can read, edit, create, and organize files. It plans its work, executes multi-step tasks, and loops you in. It's less like a chat and more like delegating to a capable colleague.

üõ†Ô∏è Do It (45 min)

Open the Claude Desktop app ‚Üí click the Cowork tab. Point it at a COPY of a real folder and try one of these:

  • "Organize these files by type and date, renaming them with descriptive names"
  • "Read all the files in this folder and create a single summary document"
  • "Look at these receipt images and create an expense spreadsheet"
  • "Review these draft documents and create a status report"
‚ö†Ô∏è Safety Rules ‚Äî Read These
  • ALWAYS use a COPY of your folder ‚Äî Cowork can delete files
  • Start with low-stakes folders (downloads, drafts) ‚Äî not financial or client docs
  • Give clear, specific instructions ‚Äî vague = risky with an agent that takes action
  • Review what Claude did before trusting the output
üí° Pro Tip

The magic of Cowork is multi-step tasks. Instead of "organize my files," try: "Read every document in this folder, group them by project, create a summary document for each project, and create a master index with links to each summary." Claude will plan the steps and execute them.
